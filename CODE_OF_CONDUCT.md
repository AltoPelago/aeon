# Code of Conduct

This repository expects respectful, good-faith participation.

## Expected behavior

- be respectful in issues, pull requests, reviews, and discussions
- assume good intent while still being precise and honest
- keep technical disagreement focused on the work, not the person
- avoid harassment, abuse, intimidation, dogpiling, or personal attacks
- avoid publishing private or sensitive information without consent

## Scope

This applies to repository spaces such as issues, pull requests, reviews, and
other project communication channels attached to this codebase.

## Enforcement

The repository is maintained under the single-maintainer model described in
[GOVERNANCE.md](./GOVERNANCE.md). The maintainer may moderate discussion,
request changes in behavior, lock conversations, or remove contributions that
do not meet these expectations.

## Reporting

If conduct in this repository crosses a line, report it through the most direct
maintainer-visible channel available for the repository context.
